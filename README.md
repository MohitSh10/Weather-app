# Weather App

A modern weather application with voice search, dark/light mode toggle, and geolocation support.

## Features

- 🌤️ Real-time weather data
- 🎤 Voice search functionality
- 🌙 Dark/Light mode toggle
- 📍 Geolocation support
- 📱 Responsive design
